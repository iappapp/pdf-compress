# PDF Compress

一个简单的 PDF 压缩 Web 应用，基于 JavaScript 实现。

## 功能
- 支持 PDF 文件上传
- 压缩 PDF 文件体积
- 下载压缩后的 PDF 文件

## 使用方法
1. 克隆本仓库：
   ```bash
   git clone <仓库地址>
   ```
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动本地服务器（可选）：
   ```bash
   npm start
   ```
4. 在浏览器中打开 `index.html`，根据页面提示上传并压缩 PDF 文件。

## 目录结构
- `index.html`：主页面
- `main.js`：前端逻辑
- `package.json`：依赖与项目信息

## 依赖

- 依赖包见 `package.json`
